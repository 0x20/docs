# Home Assistant

Home Assistant is the central automation hub for the hackerspace, integrating various systems and sensors.

## Access

- **URL:** http://ha.local
- **Login:** hsg / ask a board member for the password

## Integrations

### Lights

- **WLED LED strips:** 1t hexagons, WLED-mateo-16x16, Dig-Quad-V3
- **Bar lamps:** left and right

### Switches & Plugs

- **Sonoff devices:** Basic and Dual relays (some offline)
- **IKEA TRETAKT:** Smart plugs for various equipment
- **LED Graph TRETAKT**

### Sensors

- **Climate:** Temperature, humidity (Tasmota HTU21, DJefke sensors)
- **Light:** BH1750 illuminance sensor
- **Weather:** Local weather data, pollen levels
- **NAS (nasty):** CPU, memory, disk usage monitoring

### Printers

- **Samsung M332x:** Toner levels, status
- **Bambu 3D printer:** Status, temperatures, print progress (when online)

### Cameras

- OctoPrint camera
- Radar

### Automations

- IKEA Rodret dimmer controls
- Space state update (every 5 min)
